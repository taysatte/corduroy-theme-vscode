## üì¶ Installation

### VS Code Marketplace

1. Open **Extensions** sidebar (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for **`Corduroy`**
3. Click **Install**
4. Open **Command Palette** (`Ctrl+Shift+P` / `Cmd+Shift+P`)
5. Type `Preferences: Color Theme` and select **Corduroy**
6. Enjoy the cozies! ‚òïÔ∏è

### Command Line

