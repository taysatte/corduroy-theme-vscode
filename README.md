## Installation

1. Search `Corduroy` in VS Code <b>Extensions</b> sidebar.
2. Click <b>Install</b>.
3. Open the `Command Palette` with `CTRL+SHIFT+P` (PC) or `CMD+SHIFT+P` (MAC).
4. Apply `Corduroy` theme and enjoy the cozies! ☕️
